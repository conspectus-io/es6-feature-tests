ES6 Feature Testing

Client-Side Dependent

This is a statically hosted site used to test specific features of es6 on different clients.

Although there are es6 compatability charts online, there are a few key clients missing.

Specifically, this site aims to test its JavaScript on the following platforms:

  1. Android: 6.0 -- Native Android Browser, Chrome Mobile, Firefox Mobile, Others...
  2. Android: 7.0 -- Native Android Browser, Chrome Mobile, Firefox Mobile, Others...
  3. Others...

The major features we will test for are:

  • Basic use of let statments.
  • Basic use of arrow functions.
  • Template strings & string interpolation.
  • Basic use of classes.
